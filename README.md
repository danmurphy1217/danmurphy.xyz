## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Ruby - version 3.0.0 or later
* Jekyll - version 4.3.3 or later
* Bundler - version 2.2.0 or later

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/danmurphy1217/danmurphy.xyz.git
   ```
2. Install gems
   ```sh
   bundle install
   ```
3. Run the server
   ```sh
   bundle exec jekyll serve
   ```

## Usage

After running the server, you can access the site at `http://localhost:4000`.
